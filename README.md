# Trending-Assets
Implements a figma design, which displays the trending assets for a specific market.

This project is bootstrapped with Vite: https://vitejs.dev/

### Live demo:
https://zxcodes.github.io/trending-assets/

# Install dependencies
```npm install```

# Start dev server
```npm run dev```

# build for prod:
```npm run build```

# Start prod server:
```npm run preview```
